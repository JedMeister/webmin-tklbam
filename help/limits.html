Filesystem or Database Restore Limitations

Restore limitations can be used to control which files and databases get restored. The format is equivalent to backup overrides, but limitations only apply at restore time:

LIMIT := -?(/path/to/include/or/exclude | mysql:database[/table])

Remember, if you make a mistake you can rollback the last restore and try again.

Examples

# Ignore system configuration changes
-/etc

# Except changes to Webmin configuration
-/etc/webmin

# Exclude the Drupal cache table from being restored
-mysql:drupal6/cache
